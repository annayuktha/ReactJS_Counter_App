# ReactJS_Counter_App
Getting Started
---

This is a beginner-friendly counter application built using React and Vite. The app allows users to increment or decrement a number by clicking arrow buttons. This project showcases essential React skills such as state management and user interaction handling.

## Features
-  Adds tasks
-  Delete individual tasks
-  Moves the task up and down
-  Live state updates using `useState`



## Tech Stack

- Increment the counter
- Decrement the counter
- Real-time updates using `useState`
- Fast development using Vite


To run the Project 
--
npm install
npm run dev
You will receive a link to which you can run the application. For ex: http://localhost:port_number/
